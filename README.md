# General News Portal 🌍

A dynamic and responsive news website that fetches real-time world news using the NewsAPI. This project is part of my software engineering portfolio.

## 🚀 Features
- **Real-time News:** Fetches the latest headlines globally.
- **Categories:** Explore news in Tech, Politics, and Sports.
- **Responsive Design:** Optimized for both mobile and desktop views.
- **Developer Profile:** A dedicated "About" page featuring the developer's journey.

## 🛠️ Technologies Used
- **Frontend:** HTML5, CSS3, JavaScript (ES6+)
- **Integration:** NewsAPI (Fetch API)
- **Deployment:** GitHub Pages
- **Environment:** Termux & Spck Editor
